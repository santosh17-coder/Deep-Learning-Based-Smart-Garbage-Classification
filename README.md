# Smart Garbage Classification using Deep Learning

This project implements an intelligent waste classification system using deep learning to automatically identify different types of garbage from images. The system helps improve waste sorting efficiency and supports smart waste management and recycling initiatives.

---

## Project Overview

Manual waste segregation is inefficient and time-consuming. This project uses a deep learning model to classify garbage images into different categories automatically. A pretrained **ResNet50 convolutional neural network** with transfer learning is used to achieve accurate predictions.

The trained model is integrated with a **Flask web application** that allows users to upload an image and receive real-time waste classification results.

---

## Features

- Image-based garbage classification
- Deep learning model using ResNet50
- Transfer learning for improved accuracy
- Real-time prediction using a Flask web interface
- Supports six waste categories

---

## Waste Categories

The model classifies the following categories:

- Plastic
- Paper
- Metal
- Glass
- Cardboard
- Organic Waste
